## Setup

### 1. Clone & enter the project

```bash
git clone <your-repo-url>
cd CVProject_2026_Team13_Isometric
```

### 2. Create a virtual environment

```bash
# Windows
python -m venv venv
venv\Scripts\activate

# macOS / Linux
python -m venv venv
source venv/bin/activate
```

### 3. Install PyTorch for RTX 5050 (do this BEFORE requirements.txt)

The RTX 5050 is a Blackwell GPU (sm_120). A plain `pip install torch` installs
a CPU-only build that ignores your GPU completely. Install the GPU build first.

**Try this first (stable cu128):**
```bash
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu128
```

**Verify immediately:**
```bash
python -c "import torch; print(torch.cuda.is_available()); print(torch.cuda.get_device_name(0))"
```
Expected output: `True` then `NVIDIA GeForce RTX 5050`

**If output is False, use the nightly build instead:**
```bash
pip install --pre torch torchvision torchaudio --index-url https://download.pytorch.org/whl/nightly/cu128
```
Then verify again.

### 4. Install remaining dependencies

Only after PyTorch shows `True` above:

```bash
pip install -r requirements.txt
```

> **Windows note:** if `pycocotools` fails:
> ```bash
> pip install pycocotools-windows
> ```

### 4. Place the dataset

Unzip `gwhd_2021.zip` **one level above** the project root:

```
parent_folder/
в”њв”Ђв”Ђ CVProject_2026_Team13_Isometric/   в†ђ project
в””в”Ђв”Ђ gwhd_2021/                         в†ђ dataset here
    в”њв”Ђв”Ђ images/

    в”њв”Ђв”Ђ competition_val.csv
    в”њв”Ђв”Ђ competition_test.csv
    в””в”Ђв”Ђ metadata_dataset.csv
```

> If your dataset is elsewhere, update `DATA_DIR` in `src/config.py`.

## Running the Preprocessing Pipeline

```bash
# Run all steps (load в†’ clean в†’ EDA в†’ YOLO labels)
python main.py

# Run individual steps
python main.py --step 1   # data loading only
python main.py --step 2   # load + clean
python main.py --step 3   # load + clean + EDA
python main.py --step 4   # load + clean + YOLO label conversion
```

Output goes to `outputs/eda_plots/` and `processed_data/`.

## Running the Notebooks

Start Jupyter and open notebooks **in numbered order**:

```bash
jupyter notebook
```

| Notebook | What it does | Requires |
|---|---|---|
| `01_eda_pipeline.ipynb` | Data loading, cleaning, EDA, YOLO labels | `python main.py` OR run cells top-to-bottom |
| `02_yolov8_training.ipynb` | YOLOv8s training (50 epochs), Soft-NMS, TTA | `01` complete + GPU |
| `03_fasterrcnn_training.ipynb` | Faster R-CNN training + eval | `01` complete + GPU |
| `04_detr_training.ipynb` | DETR training + eval | `01` complete + GPU |
| `05_evaluation_ada.ipynb` | ADA metric, Grad-CAM, 3-model comparison | `02`+`03`+`04` complete |
| `06_inference_demo.ipynb` | Single-image inference + heatmap | any trained model |

## GPU Requirements

| Model | Min VRAM | Recommended batch size |
|---|---|---|
| YOLOv8s | 4 GB | 16 |
| Faster R-CNN (ResNet-50 FPN) | 6 GB | 4вЂ“8 |
| DETR | 8 GB | 2вЂ“4 |

> If running on CPU only, set `batch=2` and expect ~10Г— slower training.
